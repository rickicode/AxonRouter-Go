# Proxy Specification

> Generated by openlore v1.0.0 on 2026-07-24 01:29

## Purpose

Manages deployment of edge relay functions to serverless platforms (Vercel, Deno Deploy, Cloudflare
Workers), including source generation, platform API orchestration, deployment polling, health
testing, and pool metadata persistence.
